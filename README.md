# Gerador de Senhas

App simples em **React + Vite** para gerar senhas aleatórias, com controles de:

- **Tamanho**
- **Quantidade de dígitos**
- **Quantidade de símbolos**
- **Copiar senha** com um clique

## Como rodar

Instale as dependências:

```bash
npm install
```

Rode em modo desenvolvimento:

```bash
npm run dev
```

Build de produção:

```bash
npm run build
```
