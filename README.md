# 🔐 Gerador de Senhas React + Vite

Um projeto em React que explora o uso avançado do useEffect para criar um gerador de senhas fortes e dinâmicas. Muitas pessoas usam senhas fracas não por descuido, mas por falta de conhecimento sobre segurança básica. Este projeto visa facilitar a criação de senhas seguras de forma prática e rápida.
---

## Funcionalidades

- Gerar senhas fortes em segundos
- Ajustar tamanho e nível de complexidade
- Copiar a senha com um clique
- Atualização automática da senha ao alterar qualquer configuração

---

## Aprendizado

Durante o desenvolvimento, foquei em como o useEffect pode reagir a mudanças de estado:

- Antes: gerava a senha apenas ao clicar no botão
- Depois: usando useEffect, a senha é regenerada automaticamente sempre que o usuário altera tamanho ou complexidade

Isso melhorou a experiência do usuário e aprofundou meu entendimento sobre reatividade em React.
